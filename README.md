데모 사용법은 [http://www.oracloud.kr/post/oracle_code_provisioning/](http://www.oracloud.kr/post/oracle_code_provisioning/)을 참조하시기 바랍니다.

데모에 사용된 툴은 다음과 같습니다.

- Terraform
- Ansible
- Testinfra
- Vagrant
